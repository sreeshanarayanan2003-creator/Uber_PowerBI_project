# Uber_PowerBI_project
Power BI Dashboard analyzing Uber trip data to identified demand tracks, peak hours and ride patterns.

*Uber Data Analytics Project – Power BI*

Overview

This project focuses on analyzing Uber ride data to gain insights into trip patterns, demand trends, and overall business performance. The objective is to transform raw ride data into meaningful visual insights that can support operational and strategic decision-making.
The project demonstrates practical data analytics skills using Power BI for visualization and MS Excel for data preparation, along with professional reporting and presentation.

Dataset

The dataset contains Uber ride details such as trip dates, pickup and drop-off locations, trip counts, and other ride-related metrics.

Data is used for analytical and learning purposes.

The raw dataset was cleaned and structured before visualization.


Tools & Technologies

Power BI – Data modeling, DAX measures, and dashboard creation

MS Excel – Data cleaning, preprocessing, and validation

Gamma – Presentation (PPT) creation for insights storytelling


Project Steps

1. Collected and reviewed the Uber dataset in MS Excel


2. Cleaned the data by removing duplicates and handling missing values


3. Prepared data structure suitable for analysis


4. Imported data into Power BI


5. Created calculated measures and KPIs


6. Designed an interactive Power BI dashboard


7. Created a report and presentation using Gamma to summarize insights
